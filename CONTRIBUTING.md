# Contributing

## Workflow
1) `git checkout -b feat/your-feature`
2) Small commits, clear messages
3) Open PR with screenshots

## A11y & Brand
- High contrast, clear focus states
- Gold + grayscale only
- Test mobile + desktop (3D is desktop-only)

## Review checklist
- [ ] No console errors
- [ ] Lighthouse 90+
- [ ] Docs updated
